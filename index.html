Installation du script

Il vous faut une extension qui permet d'installer des script :

Si vous en possédez dèjà une passez à l'étape suivante

Chrome : TamperMonkey

Firefox : TamperMonkey

Opera : TamperMonkey

Safari : TamperMonkey

Android( seulement sur firefox ) : Firefox et et le plugin 'USI' https://addons.mozilla.org/fr/android/addon/userunified-script-injector/?src=search

(Si vous utilisez un autre navigateur cherchez un plugin de type Tampermonkey/greasemonkey sur Google et avec un peu de chance vous trouverez) ensuite il vous suffit de cliquer ici le script est sur https://openuserjs.org/


Télecharger le script

Ensuite cliquer sur ce lien

Et cliquez sur Installer
